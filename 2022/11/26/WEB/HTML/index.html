HTML 学习

教程 ： https://www.runoob.com/html/html-tutorial.html
参考手册 ： https://www.runoob.com/tags/html-reference.html
速查列表：https://www.runoob.com/html/html-quicklist.html || https://www.runoob.com/html/html-tag-name.html

超文本标记语言（英语：HyperText Markup Language，简称：HTML）是一种用于创建网页的标准标记语言。

HTML网页结构

HTML 元素

  • 元素以 开始标签（起始标签 Opening tag）起始，以 结束标签（闭合标签 Closing tag）结束
  • 元素的内容书写在开始标签与结束标签之间
    • 某些元素（空元素）具有空内容，其以开始标签的结束而结束
  • 元素可以用于属性
  • 元素之间可以相互嵌套

HTML 属性

  • 属性是HTML元素提供的附加信息，一般书写在开始标签中，且总是以 名称-值 对的形式出现
    • 例如 HTML链接元素又标签 <a> 定义，其链接的地址由 href属性指定
  • 属性值应当被包括在 双引号 中，如果属性值的文本包括双引号，则可以使用 单引号 包围
  • 推荐使用小写字母书写 书写名称

HTML 常用标签

标题 <h1></h1>

用于定义内容标题，共有<h1>-<h6>6个可用，注意浏览器会自动在标题前后添加空行，同时搜索引擎也是使用标题来为网页确定结构与编制索引

水平线 <hr>

这是一个空元素标签，用来创建水平线，分隔内容

注释

段落<p></p>

用于定义一个内容段落，属于块级元素，浏览器将自动在段落前后添加空行。注意浏览器将简化段落内容中的排版信息，如果需要换行，需要注明换行标签<br>

文本格式化标签

其他格式化文本标签 https://www.runoob.com/html/html-formatting.html

超文本链接

  • target 属性，用于指定被链接文档的打开方式
  • id 属性，用于创建一个HTML文档书签，它不会显示。可以在其他地方来引用以访问定义该属性的位置
  • 样式

头部元素 <head>

用于定义HTML头部，在其中可以插入脚本（scripts）, 样式文件（CSS），及各种meta信息。
可以添加在头部区域的元素标签为:
<title> 定义文档标题，必须的元素，这个标题将作为浏览器工具栏显示标题，收藏夹标题，搜索引擎结果页面标题
<style> 定义了样式文件的引用地址，也可以直接在该标签内容中指定定义样式
<meta> 描述基本的元数据，通常用于指定网页的描述，关键词，文件的最后修改时间，作者，和其他元数据。通常配合 name content 两个属性来定义元数据的数据名与内容
<link> 定义文档到外部资源之间的关系，常用于链接到样式表
<script> 用于加载脚本文件
<base> 描述一个基本的链接地址，其定义方式与属性基本同 <a>，这个标签的定义将作为所以链接标签的默认链接

图像标签 <img>

图像标签<img>定义HTML中的图像，它是一个空标签

属性 src 指定了图像的储存位置，名为”image.jpg”的图像应当被存储在网站对应的image文件夹中，属性 alt 指定了图像无法被载入时的替换显示文本
属性 width 与 height 可以指定图像的宽高

  • <map>标签配合img标签使用，用于创建可点击区域的图像映射
    使用方法为：
  1. 在 <img> 标签中通过 usemap属性指定所使用的map
  2. 定义一个map并使用name属性来和img连接

表格标签 <table>

表格由 <table> 标签来定义。每个表格均有若干行（由 <tr> 标签定义），每行被分割为若干单元格（由 <td> 标签定义）。字母 td 指表格数据（table data），即数据单元格的内容。

列表标签

  • 无序列表 <ul>
    可以使用属性 style 来指定不同的表项
  • 有序列表 <ol>
    可以使用属性 type 来指定不同的序列号书写方式
  • 自定义列表 <dl>

HTML 区块

  • 如<h1> <p>``<ul>``<table>等元素在显示时会以新的一行来显示和结束，这样的元素被称为 块级元素
  • 如<b>``<td>``<a>``<img>等元素不会以新的一行来显示，这样的元素被称为 内联元素
  • 块级元素 <div> 的一个特殊的块级元素，它没有特殊的定义，专门用于组合其他的HTML的元素，是一个容器。一般可用于进行文档布局，以及配合CSS一同使用时可以对于较大的内容块设置样式
  • 内联元素 <span>是一个特殊的内联元素，它没有特殊的定义，可以作为文本的容器，同样可以配合CSS来使用

HTML 表单

  • 表单<form>是一个包含表单元素的区域，表单元素是允许用户在表单中输入内容，如文本、下拉选择、单选、复选等

HTML 框架 <iframe>

  • 通过使用 <iframe src="URL"></iframe> 可以在当前页面嵌套别的网页
  • loading属性可以指定不同的加载模式，width height可以指定该元素的大小
  • 出于有些网页不希望被嵌套, 响应头中有一选项：X-Frame-Options，有三个可配置值
    • DENY：表示该网站页面不允许被嵌套，即便是在自己的域名的页面中也不能进行嵌套。
    • SAMEORIGIN：表示该页面可以在相同域名页面中被嵌套展示。
    • ALLOW-FROM uri：表示该页面可以在指定来源页面中进行嵌套展示。

脚本标签 <script>

  • <script> 标签用于定义客户端脚本，比如 JavaScript
  • 既可包含脚本语句，也可通过 src 属性指向外部脚本文件
  • JavaScript 最常用于图片操作、表单验证以及内容动态更新
    [JavaScript教程] https://www.runoob.com/js/js-tutorial.html
  • noscript 标签提供无法使用脚本(浏览器禁用脚本，或浏览器不支持客户端脚本)时的替代内容

字符实体

  • 在HTML中预留了一些字符作为保留字，如果希望正确的显示这些字符，需要使用字符实体代替
  • 字符实体的书写方式为 &entity_name 或者 &#entity_number
  • 其他如下情况，也需要使用到字符实体
    • 不间断的空格 - &nbsp - 这是因为HTML会自动简化，将多个连续空格简化显示为一个
    • 音标符号 - 因为键盘打不出这些符号
    • 一些其他符号

统一资源定位器 URL

统一资源定位器 Uniform Resource Locator 是一个网页地址，其组成为
schmem://host.domain:port/path/filename

  • scheme - 定义因特网服务的类型

    • 可选 ： http - 超文本传输协议，不加密；https-安全超文本传输协议-加密; ftp-文件传输协,用于将文件下载或上传至网站; file-您计算机上的文件。

  • host - 定义域主机（http 的默认主机是 www）

  • domain - 定义因特网域名，比如 runoob.com

  • :port - 定义主机上的端口号（http 的默认端口号是 80）

  • path - 定义服务器上的路径（如果省略，则文档必须位于网站的根目录中）。

  • filename - 定义文档/资源的名称

  • 注意：URL 只能使用 ASCII 字符集，当 URL 包含 ASCII 集合之外的字符，URL 必须转换为有效的 ASCII 格式。URL 编码使用 “%” 其后跟随两位的十六进制数来替换非 ASCII 字符。URL 不能包含空格。URL 编码通常使用 + 来替换空格。

HTML 样式 - CSS

  • CSS (Cascading Style Sheets) 用于渲染HTML元素标签的样式
  • 其有三种定义方式：
      1. 内联样式 - 在标签中通过 style 属性中定义
      1. 内联样式表 - 在 head 标签中的 style标签 来包含 CSS
      1. 外部引用 - 在 head 标签中的 link 标签 使用外部的 CSS 文件
