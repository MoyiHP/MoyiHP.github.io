CSS

CSS (Cascading Style Sheets，层叠样式表），是一种用来为结构化文档（如 HTML 文档或 XML 应用）添加样式（字体、间距和颜色等）的计算机语言

语法规则

  • CSS 规则由两个主要的部分构成：选择器，以及一条或多条声明:
    选择器 {属性：值; 属性：值； ... }
  • 注释
    /* 这是一条注释 */

使用方法

外部样式表(External style sheet)

当样式需要应用于很多页面时，外部样式表将是理想的选择。在使用外部样式表的情况下，你可以通过改变一个文件来改变整个站点的外观。每个页面使用 <link> 标签链接到样式表

这是一个示例的CSS外部样式表

内部样式表(Internal style sheet)

当单个文档需要特殊的样式时，就应该使用内部样式表，可以使用 <style> 标签在文档头部定义内部样式表

内联样式(Inline style)

在相关的标签内使用样式（style）属性

多重样式

如果同时启用了多种样式，那么对于每个具体的元素，将会采用 继承-覆盖 的原则，即按照 外部-内部-内联 的顺序继承，如果对于某个 css 属性都有定义，那么采用后者覆盖前者的方式确定最终的属性值。
即优先级为 （内联样式）Inline style > （内部样式）Internal style sheet >（外部样式）External style sheet > 浏览器默认样式

选择器

选择器名不要以数字开头，它无法在 Mozilla 或 Firefox 中起作用。

ID选择器

  • id 选择器可以为标有特定 id 的 HTML 元素指定特定的样式
  • HTML元素以id属性（唯一）来设置id选择器,CSS 中 id 选择器以 “#” 来定义

类选择器

  • class 选择器用于描述一组元素的样式，class 选择器有别于id选择器，class可以在多个元素中使用
  • class 选择器在 HTML 中以 class 属性（可多个元素共有）表示, 在 CSS 中，类选择器以一个点 . 号显示
  • 可以指定特定的 HTML 元素使用 class

分组与嵌套

分组
用于减少重复代码

嵌套
用于细化定义

  • p{ }: 为所有 p 元素指定一个样式
  • .marked{ }: 为所有 class=”marked” 的元素指定一个样式
  • .marked p{ }: 为所有 class=”marked” 元素内的 p 元素指定一个样式
  • p.marked{ }: 为所有 class=”marked” 的 p 元素指定一个样式

组合选择符

在 CSS3 中包含了四种组合方式:

  1. 后代选择器(以空格 分隔)
    后代选择器用于选取某个元素的后代元素，简单来说就是包含在该类元素中某类元素
    如 div p选择的是包含在 div 中的 p元素

  2. 子元素选择器(以大于 > 号分隔）
    子代选择器基本类似后代选择器，但是子代选择器只能选择直接包含在该类元素中的某类元素
    如div>p选择的是直接包含在div中的p元素

  3. 相邻兄弟选择器（以加号 + 分隔）
    选择紧接在另一元素后的元素，且二者有相同父元素
    div+p 选取了所有位于 <div> 元素后的第一个 <p> 元素

  4. 普通兄弟选择器（以波浪号 ～ 分隔）
    选取所有指定元素之后的相邻兄弟元素
    div~p选取了所有 <div> 元素之后的所有相邻兄弟元素 <p>

CSS盒子模型

  • 所有HTML元素可以看作盒子，在CSS中，”box model”这一术语是用来设计和布局时使用
  • CSS盒模型本质上是一个盒子，封装周围的HTML元素，它包括：边距，边框，填充，和实际内容
  • 盒模型允许我们在其它元素和周围元素边框之间的空间放置元素
    • Margin(外边距) - 清除边框外的区域，外边距是透明的
    • Border(边框) - 围绕在内边距和内容外的边框
    • Padding(内边距) - 清除内容周围的区域，内边距是透明的
    • Content(内容) - 盒子的内容，显示文本和图像
  • width 和 height 设置的是内容区域的宽度和高度

CSS属性

背景

  1. 背景颜色： 属性名 background-color 值：一个采用十六进制编码的演示
  2. 背景图像：属性名 background-image 值：url(图像文件)
  • 背景图像重复：属性名 background-repeat 值：repeat-x | no-repeat
  • 背景图像位置（起始）：属性名 background-position 值：right top …
  • 背景图像是否根据页面滚动：属性名 background-attachment

  • 简写时的顺序为：background-color background-image background-repeat background-attachment background-position

文本

  1. 文本颜色：属性名 color
  2. 文本对齐方式：属性名 text-align
  3. 文本修饰（用于设置或删除文本的装饰，如链接文本的特殊装饰）：属性名 text-decoration
  4. 文本转换：来指定在一个文本中的大写和小写字母，可用于所有字句变成大写或小写字母，或每个单词的首字母大写
    属性名： text-transform
  5. 文本缩进：属性名 text-indent
  6. 文本间距：属性名 letter-spacing
  7. 文本行距：属性名 line-height
  8. 文本阴影：属性名 text-shadow

显示

  1. Display(显示) 与 Visibility（可见性）
  • visibility:hidden可以隐藏某个元素，但隐藏的元素仍需占用与未隐藏之前一样的空间。也就是说，该元素虽然被隐藏了，但仍然会影响布局
  • display:none可以隐藏某个元素，且隐藏的元素不会占用任何空间。也就是说，该元素不但被隐藏了，而且该元素原本占用的空间也会从页面布局中消失
  • display:inline 可以将一个元素强制改为内联 display:block可以将一个元素强制改为块

定位

  1. position 属性指定了元素的定位类型
    position 属性的五个值：
  • static ： HTML 元素的默认值，即没有定位，遵循正常的文档流对象。静态定位的元素不会受到 top, bottom, left, right影响。
  • relative：相对定位元素的定位是相对其正常位置。
  • fixed ： 元素的位置相对于浏览器窗口是固定位置。即使窗口是滚动的它也不会移动
  • absolute：绝对定位的元素的位置相对于最近的已定位父元素，如果元素没有已定位的父元素，那么它的位置相对于<html>:
  • sticky ： 基于用户的滚动位置来定位。粘性定位的元素是依赖于用户的滚动，在 position:relative 与 position:fixed 定位之间切换。它的行为就像 position:relative; 而当页面滚动超出目标区域时，它的表现就像 position:fixed;，它会固定在目标位置。
  1. z-index属性指定了一个元素的堆叠顺序

溢出

CSS overflow 属性可以控制内容溢出元素框时在对应的元素区间内添加滚动条
值 描述
visible 默认值。内容不会被修剪，会呈现在元素框之外
hidden 内容会被修剪，并且其余内容是不可见的
scroll 内容会被修剪，但是浏览器会显示滚动条以便查看其余的内容
auto 如果内容被修剪，则浏览器会显示滚动条以便查看其余的内容
inherit 规定应该从父元素继承 overflow 属性的值

浮动

  • Float 浮动，会使得元素向左或者向右移动，其周围元素相应的重新排列
  • 元素是只能水平浮动的，当一个元素浮动时，它会尽量的向左/右移动直到其外边缘碰到包含框或者另外一个浮动框的边框
  • 在浮动元素前的元素不会受到浮动元素的影响，在浮动元素之后的元素将会围绕浮动元素重新排列
  • 可以使用 “Clear” 属性来清除浮动，clear 属性指定元素两侧不能出现浮动元素。

对齐

居中

  1. 使得元素居中显示使用 margin:auto
  2. 使得元素内文本居中显示使用 text_align:center
  3. 使得图片居中对齐需要先将img元素转换为块元素，再使用margin:auto 左右对齐
  4. 使得元素左右对齐使用 position: absolute
  5. 也可以使用 float，不过当像这样对齐元素时，需要对 <body> 元素的外边距和内边距进行预定义

CSS伪元素

CSS 伪元素是用来添加一些选择器的特殊效果。

语法

常见

  1. “first-line” 伪元素用于向文本的首行设置特殊样式，只能用于块级元素

  2. “first-letter” 伪元素用于向文本的首字母设置特殊样式，只能用于块级元素

  3. CSS类配合伪元素

  4. “:before” 伪元素可以在元素的内容前面插入新内容，”:after” 伪元素可以在元素的内容之后插入新内容
