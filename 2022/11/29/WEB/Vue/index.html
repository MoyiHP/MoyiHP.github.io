Vue App

  1. 定义与作用：Vue App是一个特殊的实例对象，其可以用于对于组件的精确化控制

  2. 创建与根组件： Vue App 通过使用 createApp 函数来创建，这个函数需要传入一个组件对象以作为新创建的Vue App的根组件对象

  3. 挂载与容器：Vue App创建后需要通过 .mount() 方法挂载到一个容器上，这样这个Vue App才会得到渲染。这个容器可以是一个实际的DOM元素，也可以是一个CSS选择器字符串

  4. 关于 组件对象 在以后将使用构建流程下编写，目前的情况下可以直接书写组件模板

  5. 目前来看，组件对象其实十分类似于一个实际的 “Java 对象”，它通过 data() 函数定义了 属性，而后通过 method() 函数定义了 方法

  • data() 函数，负责提供一个方法去定义属性(格式为 属性名：默认值，….)，它将返回一个 $data 的总的属性，在 $data 属性下就是我们定义的各个属性，如 count
  • method()函数，负责提供一个方法去定义各个方法（格式为 方法名（参数表{方法体}, … ））

模板语法

【官网介绍】

即 Vue模板语法 是基于HTML语法的，是符合HTML语法规则的，其作用在于 1. 在渲染上，能够使用到之前Vue App组件实例的数据； 2. 在底层上，能转变为 JS 代码，同时最优化渲染变化所需的DOM操作

数据绑定

  1. 基本 文本插值
    在相应位置的HTML标签中，可以使用 {{属性名}}的方式将组件实例的属性以文本的形式插入到HTML文件中，这种方式一般用于显示数据

  2. 原始HTML插值
    在相应位置的HTML标签中，可以使用 <标签名 v-html="属性名">的方式，将组件实例的属性以 原始HTML文本 的形式插入到HTML文件中，这种方式一般用于插入某个HTML元素，这个新插入的元素将以相应位置的元素的子元素插入

    PS : 注意 在使用组件属性的时候，是支持使用 + - * / 等数据运算符号、split()等字符串运算函数， ？：三目运算符的

  3. Attribute 绑定
    双大括号不能在 HTML attributes 中使用
